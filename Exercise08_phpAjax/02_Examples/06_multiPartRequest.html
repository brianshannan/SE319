The purpose of this example is to show how the BROWSER sends a multi-part request to the SERVER.
We have a FORM that uses multipart/form-data encoding. We are able to send MULTIPLE files in one request. These files can be of different types (word/excel/text/pdf/image etc).
Do check that the response obtained from 06_display.php does indeed show that two files were received by the server (from the browser).
